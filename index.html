About me

Halo, perkenalkan nama Saya Akmalinnuha. Saya adalah mahasiswa semester 6 Teknik Informatika Universitas Brawijaya (UB). Ketertarikan Saya di bidang IT sudah muncul dari kelas 3 SMP. Saya mulai belajar pemrograman waktu SMA dan yang Saya pelajari adalah HTML dan Javascript. Setelah melalui berbagai tes dan riset, Saya sadar dan mantap bahwa informatika adalah passion Saya maka dari itu Saya mendaftar di program studi Teknik Informatika. Setelah melalui tes UTBK Saya diterima di Teknik Informatika Fakultas Ilmu Komputer UB. Cukup banyak hal yang Saya pelajari di teknik informatika ini, tetapi hal yang mulai Saya sadari ketika semester 4 Saya harus menetapkan di mana bidang konsentrasi Saya. Pada akhir semester 4 Saya mencoba course dicoding "Belajar Membuat Aplikasi Android untuk Pemula" dan Saya selesaikan course tersebut. Setelah menyelesaikan course tersebut Saya mulai menyadari bahwa bidang android development ternyata lebih menarik daripada web development. Dari situ Saya memilih android development sebagai konsentrasi Saya di bidang IT ini. Pengembangan android yang Saya pelajari adalah native android development dengan bahasa Kotlin. Sekarang Saya sudah yakin pada pilihan Saya, untuk menjadi android developer di masa mendatang.
